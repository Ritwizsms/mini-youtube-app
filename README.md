# Mini YouTube App

## About mini YouTube

In this mini YouTube you can search your required content and also play the video base on your search.

## Tech-Stack used

- HTML5
- CSS3
- JavaScript ( ES6 )
- YouTube API

## Features implemented

- Video will update daily on homepage
- Search your favourite content
- Play the video
